## 风控规则（v38）
1. 排除 repair_force_10d < -5%（断板修复失败）
2. 排除 高波反转 < -3%（高波动反转风险）
3. 排除 真实市值 < 20亿（小盘流动性风险）
   - 面板中`市值` = np.log(total_mv_万元)
   - 真实市值(亿) = exp(市值) / 10000
   - 过滤条件：市值 < np.log(200000) ≈ 12.21
